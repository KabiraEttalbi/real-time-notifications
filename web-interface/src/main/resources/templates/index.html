Plateforme de Notifications en Temps Réel

Bienvenue sur notre plateforme de notifications événementielle utilisant Apache Kafka et Spring Boot.

Cette plateforme permet de gérer les utilisateurs, envoyer des notifications en temps réel et surveiller tous les événements du système.

Commencer
Gestion des Utilisateurs

Inscrivez de nouveaux utilisateurs et gérez leurs informations. Chaque inscription déclenche automatiquement une notification de bienvenue.

Inscription
Notifications

Envoyez des notifications personnalisées par email, SMS ou push. Toutes les notifications sont traitées de manière asynchrone.

Monitoring

Surveillez en temps réel tous les événements de la plateforme. Visualisez les métriques et les logs d'audit.

Architecture de la Plateforme

Services Microservices
  • User Service: Gestion des utilisateurs et authentification
  • Notification Service: Traitement des notifications multi-canaux
  • Event Publisher: Publication d'événements métier
  • Logging Service: Audit et logging des événements
Technologies Utilisées
  • Apache Kafka: Broker de messages événementiel
  • Spring Cloud Stream: Abstraction pour Kafka
  • MySQL: Base de données relationnelle
  • Kubernetes: Orchestration des conteneurs
